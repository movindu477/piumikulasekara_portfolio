# Piumi Kulasekara - Portfolio

A modern, high-performance professional portfolio built with React, Vite, and Tailwind CSS.

## Features
- **Sleek Modern UI**: Custom animations and refined typography.
- **Responsive Design**: Optimized for all screen sizes from mobile to desktop.
- **Interactive Sections**: Dynamic Testimonials, Education Timeline, and a fully functional Contact area.
- **Optimized Performance**: Built with Vite 7 for lightning-fast loads.

## Deployment
This project is configured for deployment on Vercel. 
- Ensure the build command is: `npm run build`
- Ensure the output directory is: `dist`
